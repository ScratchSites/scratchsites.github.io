Quick Start Wizard 2 of 7


Wi-Fi Network Name & Passphrase
Customize the Wi-Fi network's name and the passphrase used to connect to the network.
Network Name (SSID)
The name is between 1 and 32 alphanumeric characters long. If using Automatic Band Steering this name will apply to both the radios. If Band Steering is not enabled the 5 GHz name will include this name + '-5g'.


Passphrase / Wi-Fi Security Key
The Passphrase must contain between 8 and 64 characters.

Wi-Fi Security Configuration
This wizard will configure the SBG6950AC2 to use modern WPA2-PSK security.
WPA2-PSK security is very robust when used with a strong passphrase.
Note: If older devices cannot connect using WPA2-PSK security, the SBG6950AC2 can use the legacy WEP security scheme by manually selecting WEP on the network page.
